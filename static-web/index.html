🚀 상담센터 플랫폼

안전하고 익명의 상담 서비스를 제공합니다

✅ 플랫폼이 성공적으로 실행되었습니다!

📱 주요 서비스

  • 익명 상담 - 개인정보 보호
  • 실시간 채팅 - 즉시 소통
  • 파일 공유 - 이미지/문서 전송
  • 15분 세션 - 연장 가능

🔐 보안 특징

  • 엔드투엔드 암호화 - 모든 메시지 암호화
  • 자동 데이터 삭제 - 세션 종료 시 즉시 삭제
  • 감독 모드 - 품질 향상 및 안전 확보
  • 개인정보 최소 수집 - 익명성 보장

🏗️ 기술 스택

  • 프론트엔드: Next.js 15, TypeScript, Tailwind CSS
  • 백엔드: NestJS, Socket.IO, Redis
  • 보안: AES-GCM 암호화, X25519 키 교환
  • 인프라: Docker, PWA 지원
